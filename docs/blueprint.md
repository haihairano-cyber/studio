# **App Name**: ProvaFacil

## Core Features:

- Template Creation: Create test templates with defined answer keys within the app.
- Image Upload: The ability for the user to upload a camera picture or an image with a test answers grid.
- Image Processing: Process the image to identify and extract answers.
- Grade Calculation: Calculate the grades based on the processed images compared to the answer key.
- Results Display: Display the results, showing correct/incorrect answers and the total score.

## Style Guidelines:

- Primary color: Calm blue (#64B5F6) for a trustworthy and educational feel.
- Background color: Light gray (#F0F4F8) for a clean and distraction-free interface.
- Accent color: Soft green (#A5D6A7) to highlight correct answers and positive feedback.
- Body and headline font: 'Inter', a sans-serif font, for clear and modern readability.
- Use clear, recognizable icons for actions like uploading images, creating templates, and displaying results.
- A clean and intuitive layout, optimized for mobile viewing, will help with the usability of the application.
- Subtle animations to give visual feedback, like when answers are being scanned, or grades are being updated.
